# parkease-frontend
## Project Description

ParkEase Frontend is the initial interface of a parking management system.
It currently implements a login system with role-based redirection and a basic registration page layout.

This project focuses on frontend structure, routing logic, and UI organization using HTML, CSS, and JavaScript.

## Pages

1. index.html (Login Page) – Allows users to log in and redirects them to the dashboard.

2. dashboard.html – Main dashboard page after successful login.

3. register.html – Registration page layout (partially styled).

## How to Run

1. Clone the repository: git clone https://github.com/roscalyse/parkease-frontend

2. Open index.html in your browser.

3. (No server required — runs directly in the browser.)

4. login with any credentials (No authentication implemented yet)

## Features Implemented

 - Login form

 - Project folder structure organized

 - Dashboard

 - Initial registration page layout (partially styled)
